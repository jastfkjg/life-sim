# Life Sim - 人生模拟器

基于LLM的文字类模拟人生游戏，每一次选择都会影响你的人生轨迹。

## 游戏特性

### 10维属性系统

| 属性 | 说明 | 归0后果 |
|------|------|--------|
| 💚 健康 | 身体状态 | **死亡** |
| 💰 金钱 | 财务状况 | 破产但不死 |
| 😊 快乐 | 心理状态 | 痛苦但不死亡 |
| 📈 事业 | 职业发展 | 失业但不死亡 |
| 🤝 社交 | 社交能力 | 封闭但不死亡 |
| 👨‍👩‍👧 家庭 | 结婚/子女 | 孤独但不死亡 |
| 👯 朋友 | 友谊深度 | 孤立但不死亡 |
| 💕 爱情 | 感情经历 | 空白但不死亡 |
| 🌙 孤独 | 内心感受 | 越低越好 |
| 🦅 自由 | 自主程度 | 越低越被束缚 |

### 属性联动

- 💰 金钱低 → 💚 健康下降（营养不良）
- 💚 健康低 → 😊 快乐下降
- 🤝 社交低 → 🌙 孤独上升
- 🌙 孤独高 → 😊 快乐下降
- 🦅 自由高 → 🌙 孤独上升（高处不胜寒）
- 👨‍👩‍👧 家庭高 → 🦅 自由下降（责任束缚）
- 💕 爱情高 → 😊 快乐上升
- 👯 朋友高 → 🌙 孤独下降

### 事件类型

LLM会根据当前状态生成多样化事件：
- 🎯 机会类（升职、投资、邂逅）
- ⚠️ 挑战类（疾病、纠纷、危机）
- 💝 人际类（婚礼、友情、背叛）
- 🎲 选择类（留学、创业、搬迁）
- ❓ 意外类（彩票、车祸、遗产）

### 人生成就

- 💍 结婚
- 👶 为人父母
- 💰 财务自由
- 🌙 孤独终老

## 技术栈

- 前端：原生 HTML/CSS/JavaScript（无框架）
- 后端：Python Flask
- LLM：支持 OpenAI / MiniMax / DeepSeek

## 快速开始

```bash
# 1. 进入目录
cd /root/life-sim

# 2. 配置API Key
cp .env.example .env
# 编辑 .env 填入你的 API key

# 3. 安装依赖
pip install -r requirements.txt

# 4. 启动服务器
python server.py

# 5. 打开浏览器
# http://localhost:5000
```

## 游戏流程

1. **创建角色** - 输入名字、选择性别、设定起始年龄
2. **生成背景** - LLM为你创造独特的身世
3. **做出选择** - 每个回合LLM生成事件，提供3个选项
4. **属性变化** - 选择影响各项属性，属性间互相影响
5. **人生终章** - 健康归0死亡、80岁自然结束、或快乐归0结束
6. **总结评分** - 查看一生轨迹、成就、和最终评分

## 评分系统

最终得分 = 10项属性（不含孤独）的平均值（孤独越高越扣分）

满分为100分，评价等级：
- 90+ 传奇人生
- 70-89 精彩人生
- 50-69 普通人生
- 30-49 坎坷人生
- <30 悲剧人生
